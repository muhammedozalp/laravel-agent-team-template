# Backlog — undecided ideas

_Git-tracked parking lot. An idea lands here as `<slug>.md` (unnumbered — "should
we?"); when the owner greenlights it, it **graduates** to `../feature/NN-slug.md`
(numbered, checklist — "yes, do it") using `../templates/feature-spec.md`, and the
backlog file is deleted in the same PR._

Keep entries short: what the idea is, why it might matter, open questions. Anything
already decided doesn't belong here — that's a feature spec or an ADR.
